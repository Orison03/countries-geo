# Cities Data

<Steps>
### Get All Cities

This gives you an array of all cities.

<Tabs items={['ES6 Module', 'AMD Module']}>
  <Tabs.Tab>
    ```ts filename="cities.ts" showLineNumbers copy
    import { Cities } from 'countries-geo';  

    console.log(Cities.getAll); 
    // Logs all Cities (e.g., [{ iso: "GH-AA-ACH", name: "Achimota" }, ...])
    ```
  </Tabs.Tab>
  
  <Tabs.Tab>
     ```ts filename="cities.ts" showLineNumbers copy
    let Cities = require('countries-geo').Cities;

    console.log(Cities.getAll)
    // Logs all Cities (e.g., [{ iso: "GH-AA-ACH", name: "Achimota"  }, ...])
    ```

  </Tabs.Tab>
  
</Tabs>

> By default this is sorted by name.
