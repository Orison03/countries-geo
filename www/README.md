## Installation

### with Yarn
```sh filename="shell" copy
$ yarn add countries-geo
```

### with npm
```sh filename="shell" copy
$ npm i countries-geo
```

### with pnpm
```sh filename="shell" copy
$ pnpm add countries-geo
```

### with Bun
```sh filename="shell" copy
$ bun add countries-geo
````
